# 🛠️ 折叠臂 Z 轴与 AS5600 角度离线标定方案

为了让 MCU 知道“伸出 X 毫米对应的电机角度是多少”，我们需要手动采集一组离散数据来构建查找表 (LUT)。

**步骤 1：硬件准备与解耦**
* **断开动力**：拔掉 2804 电机的驱动电源（或在代码中强制设 `EN = 0`），确保机械臂可以被外力轻松推拉，不会产生闭环阻力。
* **保持逻辑通电**：确保 STM32、AS5600 磁编码器正常供电。
* **开启串口打印**：写一段极其简单的测试代码，在 `while(1)` 中以 10Hz 的频率通过串口（`BSP_UART_Printf`）打印 `DrvAS5600_GetRawAngle()` 的当前值。

**步骤 2：建立坐标零点 (Home Position)**
* 将折叠臂**完全收缩**到最极限的物理位置。
* 用卷尺或游标卡尺测量此时夹爪末端到相机基准面的真实 Z 轴垂直距离（例如：`120 mm`）。
* 记录此时串口打印的 AS5600 Raw Angle（例如：`850`）。
* 这就是 LUT 的第一组数据：`{120, 850}`。

**步骤 3：等距步进采样**
* 用手将机械臂缓慢向外拉，每移动 **20 mm** 或 **30 mm** 停一下（例如拉到 140mm, 160mm, 180mm...）。
* 在每个刻度点，记录对应的 AS5600 角度值。
* **注意观测单调性**：随着距离增加，角度值必须是持续递增或持续递减的。如果出现角度越界（比如从 4095 突变回 0），说明磁铁安装的零位不在最佳工作区内，建议重新固定 AS5600 的径向磁铁，确保整个物理行程（假设 120mm 到 300mm）落在 0-4095 的**连续区间内**。
