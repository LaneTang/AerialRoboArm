# Archive

This directory keeps historical project materials that are useful for traceability but are not part of the current portfolio-level entrance.

## Contents

| Directory | Contents |
| --- | --- |
| `notes/` | Meeting notes, branch summaries, early control bring-up plans, mock data, and development logs. |
| `interface-drafts/` | Early vision / host-side protocol drafts and GUI planning notes. |
| `legacy-assets/` | Older figures that were previously stored under `Doc/`. |
| `driver-mirror/` | A historical duplicated mirror of HAL/CMSIS files that previously lived under `Doc/Drivers`. The active vendor dependencies are under the repository-root `Drivers/` directory. |

The active project entrance is the root README plus the curated documents listed in [Doc/README.md](../README.md).
