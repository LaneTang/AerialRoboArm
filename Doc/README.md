## PID 电机控制

目标角度 θ_target
↓
PID 计算 → 输出 PWM 值（-1000 ~ 1000）
↓
电机驱动 → 转速 ω ∝ |PWM|
↓
机械臂关节转动 → 实际角度 θ_feedback（编码器读取）
↓
反馈给PID → 误差 e = θ_target - θ_feedback
